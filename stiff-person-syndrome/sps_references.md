# References — Stiff Person Syndrome (SPS) QSP Model

review references underpinning the QSP map, the mrgsolve ODE structure,
and the Shiny scenario library. All PubMed links use the canonical
`https://pubmed.ncbi.nlm.nih.gov/` form.
