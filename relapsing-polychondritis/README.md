## 개요 (Overview)

재발성 다발연골염(RPC)은 귀·코·기관·기관지·관절·눈의 연골과 탄성 조직을 표적으로 하는 희귀 자가면역 질환으로, 연간 발생률은 약 3.5/100만 명입니다. II형·IX형·XI형 콜라겐 및 마트릴린-1에 대한 자가항체와 CD4+ T세포 반응이 연골 기질을 파괴하며, 기도 연골 침범 시 기관 협착으로 생명을 위협합니다. 코르티코스테로이드가 급성기 핵심 치료이며, 재발 억제를 위해 다파손·메토트렉세이트·아자티오프린 등 면역억제제, 난치성 병례에서 항TNF·항IL-6·항CD20 생물학제제가 사용됩니다.

## 핵심 병태생리 경로 (Key Pathways)

| 경로 | 핵심 분자/기전 | 임상 결과 |
|------|----------------|-----------|
| 콜라겐 자가항원 인식 | II/IX/XI형 콜라겐 항원 → 자가항체(IgG) + CD4+ Th1 활성화 | 연골 면역 손상 개시 |
| Th1/Th17 침윤 | IFN-γ·IL-17 → 연골막 대식세포·호중구 동원 | 연골 기질 분해, 급성 염증 |
| MMP 매개 연골 분해 | TNF·IL-1β → MMP-1·3·13·ADAMTS-5 분비 | 연골 기질(프로테오글리칸·콜라겐) 소실 |
| 기도 연골 침범 | 성문하·기관 연골 염증 → 섬유화·연화 | 기도 협착, 호흡 부전 위험 |
| 내이·안구 혈관 염증 | 내이 혈관염 → 감각신경성 청력 소실; 공막염·포도막염 | 감각기관 손상 |
| 보체 활성화 | 자가항체 → C1q·C3a·C5a 활성화 → 연골 내 호중구 유입 | 연골 파괴 가속 |
| 섬유화·재형성 | TGF-β → 섬유아세포 활성화 → 기도 섬유화 | 비가역적 기도 변형 |

## 주요 약물 표적 (Drug Targets)

- **코르티코스테로이드 (프레드니솔론)**: NF-κB 억제 → 광범위 항염; 급성 발작 1차 치료, 기도 침범 시 고용량 필수
- **다파손 (Dapsone)**: 호중구 이동·MPO 억제; 경증 재발 억제, 피부·이개 연골 발작 예방
- **메토트렉세이트**: 엽산 길항 → T세포 증식 억제; 스테로이드 절감 목적
- **아자티오프린/마이코페놀레이트**: 퓨린 합성 억제; 유지 면역억제
- **항TNF 제제 (인플릭시맙, 에타너셉트)**: TNF 차단 → MMP·IL-1β 감소; 난치성 기도·관절 침범
- **항IL-6R (토실리주맙)**: IL-6 신호 차단; 전신 염증 지표 정상화
- **항CD20 (리툭시맙)**: B세포 고갈 → 자가항체 감소; 중증 재발 병례

## mrgsolve 모델 (ODE Model)

- **구획 구조**: 스테로이드/다파손/메토트렉세이트/생물학제제 PK 구획 + 자가항체 역가 동태, Th1 활성화 지수, MMP 생성 구획, 연골 두께 손상 구획, 기도 직경 협착 예측 모듈 포함
- **주요 치료 시나리오**: 무치료 자연 경과, 프레드니솔론 단독, 다파손, 프레드니솔론+메토트렉세이트, 프레드니솔론+아자티오프린, 인플릭시맙, 토실리주맙, 리툭시맙, 스테로이드+생물학제제 병용 등 다수
- **보정/근거**: McAdam 진단기준 관련 임상 코호트 데이터, Mathian 등 토실리주맙 사례군 연구, Moulis 등 프랑스 전국 코호트 문헌 기반

## Shiny 대시보드 (Dashboard)

6개 탭으로 구성: (1) **환자 프로파일** — 침범 부위(귀·코·기도·관절·안구)·자가항체 상태 설정; (2) **PK 프로파일** — 스테로이드/면역억제제/생물학제제 혈중 농도; (3) **PD 주요지표** — 자가항체·MMP·IL-1β 억제 동태; (4) **임상 엔드포인트** — 이개 발작 횟수, 기도 직경, 청력, 안질환 활성도; (5) **시나리오 비교** — 치료 전략 간 발작 빈도·장기 손상 비교; (6) **바이오마커** — CRP, ESR, II형 콜라겐 항체, 폐기능(FEV1).
