## 질환 개요 (Disease Overview)

**혈전성 혈소판감소성 자반증(TTP)**은 ADAMTS13(von Willebrand factor 절단 금속단백분해효소)에 대한 자가항체에 의해 발생하는 희귀 혈전성 미세혈관병증(TMA)입니다.

- **병태생리 핵심**: 자가항체 → ADAMTS13 활성 억제 → 초거대 VWF 다량체(ULVWF) 축적 → 미세혈전 형성 → 혈소판감소증 + 미세혈관병증 용혈성 빈혈(MAHA) + 다장기 허혈
- **발생률**: 약 3–7명/100만 명/년; 여성:남성 ≈ 3:1; 30–50대 호발
- **치료 전 사망률**: ~90% → **현재 TPE 기반 치료**: ~10–20%

### 진단 3요소 (Thrombotic Microangiopathy Triad)
1. **혈소판감소증** (PLT <30 × 10⁹/L)
2. **MAHA** (쿰스 음성 용혈성 빈혈: LDH ↑, 파편적혈구 ↑, 합토글로빈 ↓)
3. **ADAMTS13 결핍** (<10 U/dL = %)
