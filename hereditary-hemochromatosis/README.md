## 질환 개요 (Disease Overview)

**유전성 혈색증(Hereditary Hemochromatosis, HH)**은 **HFE 유전자 돌연변이(C282Y/H63D)**로 인해 헵시딘(hepcidin) 생성이 감소하고, 그 결과 소장에서 철 흡수가 과도하게 증가하여 간·심장·췌장·관절·뇌하수체에 철이 축적되는 상염색체 열성 유전 대사질환입니다.

| 특성 | 내용 |
|------|------|
| **유병률** | 북유럽계 1/200–400명 |
| **주요 유전자** | HFE (chr 6p21.3) |
| **핵심 돌연변이** | C282Y (~85%), H63D (~15%), S65C (희귀) |
| **침투도** | 15–50% (생화학적); ~10% (임상 표현) |
| **총 체내 철** | 정상 3–5 g → HH 20–40 g |
| **진단 기준** | TSAT >45% + 혈청 페리틴 상승 → HFE 유전자형 검사 |
| **치료** | 사혈(phlebotomy) — 표준, 철 킬레이트제(DFO/DFX/DFP) |
