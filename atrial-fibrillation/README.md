# 심방세동 (Atrial Fibrillation, AF) — QSP Model

## 개요

심방세동(AF)은 전 세계 성인 인구의 약 1–2%에서 발생하며, 65세 이상에서는 5–10%에 달하는 가장 흔한 지속성 부정맥입니다. 뇌졸중, 심부전, 인지장애, 사망의 독립적 위험인자로서, 연간 의료 비용이 미국에서만 260억 달러를 초과합니다.

| 항목 | 내용 |
|------|------|
| **ICD-10** | I48 |
| **유병률** | 전 세계 3,300만 명 (2010년 기준) |
| **뇌졸중 위험** | AF 없는 경우 대비 5배 증가 |
| **5년 사망률** | 심부전 동반 시 ~50% |
| **핵심 치료 전략** | 율동 조절 vs 맥박수 조절 + 항응고 치료 |

## 병태생리 핵심 경로 (Key Pathophysiological Pathways)

### 1. 발생 기전 (Initiation)

| 기전 | 핵심 분자/구조 | 임상 결과 |
|------|--------------|---------|
| 폐정맥 이소성 박동 | PV sleeve 근육 → spontaneous depolarization | AF 촉발 트리거 (85-95%) |
| 칼슘 과부하 | RyR2 기능장애 → 지연후탈분극 (DAD) | 촉발활동 (triggered activity) |
| 조기후탈분극 (EAD) | ICaL 재활성화 → IKr 차단 시 | 특히 서맥 조건에서 발생 |
| 자율신경 트리거 | 교감/미주신경 동시 활성화 | 이른 아침 AF 발생 패턴 |
| 신장 활성화 채널 | SAC (stretch-activated channels) | 심방압력 증가 → AF |

### 2. 전기적 리모델링 (Electrical Remodeling — "AF begets AF")

| 변화 | 기전 | 결과 |
|------|------|------|
| AERP 단축 | ICaL 하향조절 (Cav1.2↓) + IK1 상향조절 | 파장 단축 → 회귀 유지 |
| 전도 속도 저하 | Cx40/Cx43 분포 이상 | 전기적 불연속성 증가 |
| 속도 의존성 적응 소실 | IKs 하향조절 | AERP rate-dependence 감소 |
| 이온 채널 재분포 | IKACh 구성적 활성화 | 정상 박동 복원 어려움 |

### 3. 구조적 리모델링 (Structural Remodeling — Fibrosis)

| 신호 경로 | 매개체 | 최종 변화 |
|----------|--------|---------|
| Ang II 과활성 | TGF-β1 → Smad2/3 → 섬유아세포 활성화 | 심방 간질 섬유화 |
| 산화 스트레스 | NADPH 산화효소 → ROS | 미토콘드리아 기능장애 |
| NLRP3 인플라마솜 | IL-1β, IL-18 방출 | 만성 심방 염증 |
| EAT 분비 물질 | FGF-2, IL-6, TGF-β | 심외막 지방 → 섬유화 |
| 심방 확대 | LAP 증가 → 기계적 스트레스 | 좌심방 직경 >45mm |

### 4. 혈전색전증 경로 (Thromboembolism — Virchow's Triad)

| Virchow's Triad | AF에서의 구현 | 임상 결과 |
|----------------|------------|---------|
| 혈액 정체 (Stasis) | 좌심방 이(LAA) 내 느린 혈류 | LAA 혈전 형성 |
| 내피 기능장애 | 심방 내피 vWF↑, TF 발현 | 혈소판 활성화 |
| 혈액과응고 상태 | 피브리노겐↑, D-dimer↑, 트롬빈 생성↑ | 혈전 형성 촉진 |

## 약물 PK/PD 파라미터 (Drug PK/PD Parameters)

### 율동 조절 (Rhythm Control)

| 약물 | 분류 | 주요 PK | 주요 기전 | 임상 효과 |
|------|------|---------|---------|---------|
| **아미오다론** | Class III | t½=40–55일, Vd=60L/kg | INa/ICaL/IKr/IKs/β 차단 | 1년 동율동 유지율 65% |
| **드로네다론** | Class III* | t½=13–19h, F=15%(고지방식) | 멀티채널 차단 (비요오드) | ATHENA: CV 사망/입원 24%↓ |
| **플레카이니드** | Class Ic | t½=12–27h, F=85–90% | INa 강력 차단 (use-dependent) | 구조적 심장질환 없는 경우만 |
| **프로파페논** | Class Ic | t½=2–10h (CYP2D6 의존) | INa/ICaL/IKr + β 차단 | 동율동 전환율 ~57% |
| **소탈롤** | Class III + β | t½=12h, 신장 배설 | IKr 차단 + β-blocker | TdP 위험 모니터링 필요 |
| **도페틸리드** | Class III | t½=10h, CrCl 의존 | 순수 IKr 차단 | QTc 연장 모니터링 |
| **버나칼란트** | 심방선택적 | IV, t½=3h | INa/IKur 심방선택적 차단 | 동율동 전환율 50–52% |

### 맥박수 조절 (Rate Control)

| 약물 | 분류 | 주요 PK | 기전 | 목표 심박수 |
|------|------|---------|------|------------|
| **메토프롤롤** | β1-blocker | t½=3–7h, F=40%, CL=65L/h | β1 차단 → AV 전도 억제 | 안정 시 <110 bpm |
| **딜티아젬** | Non-DHP CCB | t½=3.5–5h, F=40% | ICaL → AV 전도 억제 | 운동 시 <130 bpm |
| **디곡신** | 강심배당체 | t½=36–48h, 신장 배설 | Na/K ATPase → 미주신경↑ | 만성 HF 동반 시 우선 |
| **이바브라딘** | If 차단제 | t½=2h, CYP3A4 | If (HCN4) 차단 → HR↓ | 부정맥(AF)에는 비적응증 |

### 항응고 치료 (Anticoagulation)

| 약물 | 표적 | 주요 PK | 뇌졸중 감소 | 출혈 위험 |
|------|------|---------|------------|---------|
| **와파린** | Vitamin K Cycle (VKOR) | t½=40h, CYP2C9/2C19 | 64% vs 위약 | INR 조절 필요 |
| **아픽사반** | Factor Xa | t½=12h, F=50%, CL=3.3L/h | ARISTOTLE: 21% vs 와파린 | 두개내 출혈 58%↓ |
| **리바록사반** | Factor Xa | t½=5–13h, F=80%(고지방식) | ROCKET AF: 비열등 | GI 출혈 약간↑ |
| **에독사반** | Factor Xa | t½=10–14h, F=62% | ENGAGE-AF: 비열등 | 출혈 18%↓ vs 와파린 |
| **다비가트란** | Thrombin (IIa) | t½=12–17h, P-gp/신장 | RE-LY: 150mg 34%↓ | 두개내 출혈 69%↓ |

## 주요 치료 시나리오 (mrgsolve 시뮬레이션)

| 시나리오 | 약물 | 주요 결과 변수 | 임상시험 근거 |
|---------|------|------------|------------|
| 1. 미치료 신규 발생 AF | — | AF 부담 증가, 섬유화 진행, 뇌졸중 위험↑ | 기저 모델 |
| 2. 맥박수 조절 | 메토프롤롤 50mg BID | 심박수 <110 bpm, AF 부담 불변 | RACE II |
| 3. 율동 조절 (아미오다론) | 아미오다론 200mg/day | 동율동 유지율 65%, AERP 정상화 | AFFIRM |
| 4. 항응고 단독 | 아픽사반 5mg BID | 뇌졸중 위험 21%↓ vs 와파린 | ARISTOTLE |
| 5. 맥박수 조절 + 항응고 | 메토프롤롤 + 아픽사반 | 심박수 조절 + 뇌졸중 예방 | RACE II + ARISTOTLE |
| 6. 율동 조절 + 항응고 | 아미오다론 + 아픽사반 | 동율동 유지 + 최대 뇌졸중 감소 | EAST-AFNET 4 |

## 참고 문헌 요약 (Key References)

- **AFFIRM** (2002): 율동 vs 맥박수 조절 — 사망률 차이 없음. 항응고 치료의 중요성 강조.
- **EAST-AFNET 4** (2020): 조기 율동 조절 — CV 사망/뇌졸중/입원 21%↓.
- **ARISTOTLE** (2011): 아픽사반 vs 와파린 — 뇌졸중 21%↓, 두개내 출혈 58%↓.
- **CHA₂DS₂-VASc** (2010): 점수 ≥2에서 항응고 치료 표준화.
