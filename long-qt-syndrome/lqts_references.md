# Congenital Long QT Syndrome (LQTS) — Key References for the QSP Model

> Curated bibliography supporting the mechanistic map, mrgsolve ODE model,
> and Shiny dashboard. References were selected from well-established,
> widely-cited LQTS/CiPA literature. Where the exact PMID could not be
> confirmed with full confidence, a PubMed search URL is given instead of a
> guessed numeric ID. Grouped by topic; an asterisk (★) marks **primary
> parameter/calibration anchors** for the model.
