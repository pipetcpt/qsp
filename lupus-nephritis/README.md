## 개요 (Overview)

루푸스 신염(Lupus Nephritis, LN)은 **전신 홍반 루푸스(SLE)** 의 가장 심각한 장기 합병증 중 하나로, SLE 환자의 약 40–60%에서 발생합니다. 사구체에 면역복합체(IgG/dsDNA/C1q)가 침착되면서 보체 활성화, 족세포 손상, 단백뇨, 신기능 저하로 이어지는 복잡한 병태생리를 보입니다.

Lupus Nephritis is a major cause of morbidity and mortality in SLE, requiring careful mechanistic modeling of the immune-renal axis to design effective treatment strategies.

## 기전 요약 (Key Pathophysiology)

| 경로 | 핵심 요소 |
|------|----------|
| **Type I IFN 신호** | pDC → IFN-α → BAFF ↑ → B cell hyperactivation |
| **B 세포 과활성화** | Tfh 도움 → GC → 장수 형질세포 → anti-dsDNA IgG |
| **면역복합체** | anti-dsDNA + C1q → IC → 사구체 메산지움 침착 |
| **보체 활성화** | C1q → C4 → C3 → C5a (호중구 유인) + C5b-9 (족세포 공격) |
| **족세포 손상** | 슬릿막(nephrin) 붕괴 → 단백뇨 (UPCR ≥ 0.5) |
| **섬유화** | TGF-β1 → 근섬유아세포 → 간질 섬유화 → eGFR 저하 |

## 임상 엔드포인트 정의 (Clinical Endpoints)

| 엔드포인트 | 정의 |
|-----------|------|
| **Complete Renal Response (CRR)** | UPCR < 0.5 g/g + eGFR ≥ 60 mL/min/1.73m² |
| **Partial Renal Response (PRR)** | UPCR < 1.0 g/g + eGFR ≥ 60 mL/min/1.73m² |
| **SLEDAI Renal** | Hematuria + Proteinuria + Pyuria + Casts 합계 (최대 16점) |
