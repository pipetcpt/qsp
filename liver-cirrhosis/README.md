> **QSP 질환 모델 라이브러리** · Claude Code Routine으로 자동 생성된 정량적 시스템 약리학(QSP) 모델입니다.

## 개요 (Overview)
간경변증은 만성 간 손상(바이러스성·알코올성·비알코올성 지방간·담즙정체 등)에 대한 최종 공통 반응으로, 간성상세포(HSC) 활성화에 의한 광범위 섬유화·결절 형성·정상 간 구조 소실을 특징으로 합니다. 전 세계적으로 약 1억 명이 이환되어 있으며, 연간 간 관련 사망의 주요 원인입니다. 보상성 단계에서 비보상성 단계로의 진행은 문맥압 항진(HVPG >10 mmHg)과 깊이 연관되며, 복수·정맥류 출혈·자발성 세균성 복막염·간성 뇌증·간신 증후군 등 치명적 합병증으로 이어집니다. 원인 치료(항바이러스제, 금주) 외에 합병증 관리(비선택적 베타차단제·이뇨제·리팍시민·테를리프레신)가 핵심입니다.

## 핵심 병태생리 경로 (Key Pathways)
| 경로 | 핵심 분자/기전 | 임상 결과 |
|------|----------------|-----------|
| 간성상세포(HSC) 활성화 | TGF-β1·PDGF→αSMA+ 근섬유모세포 분화 | 교원질(collagen I/III) 축적·섬유화 |
| 문맥압 항진 | 섬유화에 의한 간내 저항 증가 + 내장 혈관 확장 | HVPG 상승·정맥류 형성 |
| 간내 혈관 수축 | eNOS 감소·ET-1 증가·HSC 수축 | 간내 저항 증가 |
| 장 세균 전위 | 장 투과성 증가 → 세균 성분(LPS) 유입 | 전신 염증·SIRS |
| 신경호르몬 활성화 | RAAS·교감신경계 활성화, ADH 분비 증가 | 나트륨·수분 저류·복수 |
| 간기능 저하 | 알부민 합성·응고인자 생성·암모니아 대사 감소 | 저알부민혈증·응고장애·간성 뇌증 |
| 간신 증후군 | 신장 혈관 수축·GFR 저하 | AKI·만성 신기능 저하 |

## 주요 약물 표적 (Drug Targets)
- **비선택적 베타차단제** (프로프라놀롤·카르베딜롤): 심박출량 감소 + 내장 혈관 수축 → HVPG 감소, 정맥류 출혈 예방
- **스피로노락톤 + 푸로세미드** (이뇨제): 알도스테론 차단·소변 나트륨 배설 촉진 → 복수 조절
- **테를리프레신** (바소프레신 유사체): 내장 혈관 수축 → 정맥류 출혈·간신 증후군 치료
- **리팍시민** (비흡수성 항생제): 장내 암모니아 생성 억제 → 간성 뇌증 예방
- **항섬유화제** (로베글리타존·오베티콜산): HSC 활성화 억제, 섬유화 역전 연구 중
- **TIPS (경경정맥 간내 문맥 전신 단락술)**: 중증 복수·정맥류 출혈의 비약물 개입 모델링 포함

## mrgsolve 모델 (ODE Model)
- **구획 구조**: 약물 PK 구획(프로프라놀롤 3구획, 스피로노락톤 2구획, 테를리프레신 2구획, 리팍시민 1구획) + 질환 PD 구획(HSC 활성화, 섬유화, HVPG, 알부민·빌리루빈·INR, 복수, 알도스테론 활성, GFR, 크레아티닌, 혈중 암모니아, 간성 뇌증 등급)
- **주요 치료 시나리오**: ① 자연경과, ② 프로프라놀롤(정맥류 출혈 예방), ③ 스피로노락톤 + 푸로세미드(복수 조절), ④ 테를리프레신(간신 증후군), ⑤ 리팍시민(간성 뇌증 예방)
- **보정/근거**: BASS 시험(카르베딜롤), Baveno VII 합의, ATTIRE 시험(알부민 목표 치료) 등 주요 임상 데이터 기반

## Shiny 대시보드 (Dashboard)
환자 프로파일 입력(원인·Child-Pugh/MELD 점수·합병증 유무), 약물 PK 농도 추이, 주요 PD 바이오마커(HVPG·알부민·빌리루빈·INR), 임상 엔드포인트(복수·HE 등급·생존 예측), 치료 시나리오 비교, 합병증 위험도 시각화 탭으로 구성됩니다.

## 실행 방법 (Usage)
```r
library(mrgsolve)
mod <- mread("lc_mrgsolve_model.R")
out <- mrgsim(mod, end = 365)
plot(out)
# Shiny 대시보드:
shiny::runApp("lc_shiny_app.R")
```
```bash
# 기계론적 지도 렌더링
dot -Tsvg lc_qsp_model.dot -o lc_qsp_model.svg
```
