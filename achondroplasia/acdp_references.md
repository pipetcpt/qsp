**Note on infigratinib/PROPEL history:** an earlier draft of this model referenced a "2021 discontinuation" of the pediatric achondroplasia infigratinib program. This claim was investigated and found unsubstantiated — the PROPEL → PROPEL 2 → PROPEL 3 program instead ran continuously through a positive phase 3 readout (NEJM 2026), with an NDA planned for Q3 2026. All model files have been corrected accordingly.
