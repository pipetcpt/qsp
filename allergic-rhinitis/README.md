## 병태생리 (Pathophysiology)

알레르기 비염(AR)은 **IgE 매개 제1형 과민반응**으로, 전 세계 인구의 약 10–40%에 영향을 미치며 연간 의료비 및 생산성 손실이 수백억 달러에 달하는 대표적인 만성 알레르기 질환입니다. 꽃가루·집먼지진드기·동물 비듬·곰팡이 포자 등 흡입 알레르겐에 반복 노출 시 발생하며, 천식·알레르기 결막염·만성 부비동염과 아토피 행진(atopic march)으로 연결됩니다.

| 핵심 병리 경로 | 세부 메커니즘 |
|-------------|------------|
| **감작 단계** | 알레르겐 → 수지상세포(DC) 포획 → MHC II 제시 → Th2 분화(GATA3+) → B세포 IgE 클래스 전환 |
| **IgE-비만세포 축** | 자유 IgE → FcεRI 결합 → 비만세포 감작; 재노출 시 IgE-항원 가교결합 → 탈과립 |
| **초기 반응 (0–60분)** | 히스타민·PGD2·LTC4/D4·PAF 방출 → H1R·CysLT1R 활성화 → 재채기·콧물·소양감 |
| **후기 반응 (4–24시간)** | Th2(IL-4/5/13)·ILC2 → 호산구 유입(Eotaxin/CCL11) → 비충혈·염증 지속 |
| **상피 알라민 경로** | 알레르겐 단백분해효소 → 장벽 손상 → TSLP·IL-33·IL-25 분비 → ILC2 활성화 |
| **신경염증** | H1R → 삼차신경 활성화 → 축삭 반사 → SP/CGRP 방출 → NANC 혈관확장·분비 |
| **만성 리모델링** | IL-13→배상세포 과형성; MBP/ECP(호산구 과립 단백질)→상피 손상 → 후각 저하 |

## 모델 구조 (Model Architecture)

### 기계론적 지도 (`ar_qsp_model.dot/.svg/.png`)

12개 서브그래프 클러스터, **100개 이상 노드**:

| 클러스터 | 주요 내용 |
|---------|---------|
| 1. 알레르겐 노출 | 꽃가루/집먼지진드기/동물비듬/곰팡이, 비강 점막 접촉, 단백분해효소 활성 |
| 2. 상피 장벽/선천 알라민 | 상피세포 TSLP·IL-33·IL-25 분비, ILC2 활성화, NLRP3 인플라마솜 |
| 3. APC 감작/Th2 분화 | 형질세포양DC/골수성DC, MHC II 제시, OX40L, Th2/Th1/Treg 분화 |
| 4. B세포/IgE 클래스전환 | IgE CSR, 형질세포, 자유 IgE, ImmunoCAP 특이 IgE |
| 5. 비만세포/호염기구 활성화 | FcεRI, IgE-항원 가교, 탈과립, Syk/PLCγ 신호전달 |
| 6. 초기 염증 매개물질 | 히스타민·트립타제·헤파린·PGD2·TXA2·LTC4/D4/E4·PAF·SP·CGRP·VIP |
| 7. 수용체 활성화 | H1/H2/H4R, DP1/CRTH2, CysLT1/2R, PAFR, TRKR, α1/α2R |
| 8. 후기 반응/호산구 | IL-4/5/13/9/31, Eotaxin-1/3/MCP-4/RANTES, MBP/ECP/EPX/EDN |
| 9. 비강 생리/증상 | 혈관확장·분비선·신경·비충혈, TNSS(0-12)/RQLQ |
| 10. PK: 경구 H1 항히스타민 | 세티리진·로라타딘·펙소페나딘·빌라스틴 (흡수→혈장→H1RO) |
| 11. PK: 비강 코르티코스테로이드 | 플루티카손·모메타손·부데소니드·트리암시놀론 (GR 전좌→전사억제) |
| 12. 생물학적 제제/LTRA/면역요법 | 몬텔루카스트·오말리주맙·두필루맙·메폴리주맙·SCIT/SLIT·옥시메타졸린 |

### mrgsolve ODE 모델 (`ar_mrgsolve_model.R`)

**25개 상태 변수 (ODE 구획):**

| 구획 | 변수 | 설명 |
|------|------|------|
| 알레르겐 | `AG` | 비강 알레르겐 부하 (AU) |
| IgE | `IGE_FREE`, `IGE_MAST` | 자유 IgE, 비만세포 결합 IgE |
| 비만세포 | `MAST_ACT`, `MAST_CHG` | 활성도, 과립 충전 상태 |
| 매개물질 | `HISTAMINE`, `CYS_LT` | 히스타민, 시스테이닐 류코트리엔 |
| Th2/사이토카인 | `TH2`, `IL4`, `IL5`, `IL13` | Th2 활성, pg/mL |
| 호산구 | `EOS_B`, `EOS_N` | 혈액(cells/μL), 비강 조직 |
| 세티리진 PK | `CETI_D`, `CETI_C` | 위장관·중심 구획 |
| 플루티카손 PK | `FP_LOC` | 비강 국소 농도 (nM) |
| 몬텔루카스트 PK | `MLKT_D`, `MLKT_C` | 위장관·중심 구획 |
| 오말리주맙 PK | `OMA_D`, `OMA_C`, `OMA_P`, `OMA_IGE` | SC→2구획·IgE복합체 |

**임상시험 보정 파라미터:**
- 세티리진 Css에서 H1 수용체 점유율 ≥80% (Yanai 1995, *Allergy*)
- FP 200 μg/d → TNSS 35–40% 감소 vs 위약 (Meltzer 2005, *JACI*)
- 몬텔루카스트 10 mg/d → TNSS 20–25% 감소 (Philip 2002, *Clin Exp Allergy*)
- 오말리주맙 → 자유 IgE >95% 감소 (Fahy 1997, *AJRCCM*)
- 비강 호산구증식 FP 투여 시 ~50% 감소 (Holgate 2003, *Allergy*)

## 주요 임상시험 근거 (Key Evidence)

| 약물 | 임상시험 | 주요 결과 |
|------|---------|---------|
| 세티리진 | Yanai 1995 *Allergy* | PET 연구: Css H1-RO ≥80% |
| 플루티카손 FP | Meltzer 2005 *JACI* | TNSS -38% vs 위약 (8주, n=615) |
| 몬텔루카스트 | Philip 2002 *Clin Exp Allergy* | 계절성 AR TNSS -22% |
| 오말리주맙 | Meltzer 2000 *JACI* | 다년성 AR 비증상 점수 개선 |
| 두필루맙 | Bachert 2016 *NEJM* | 비용종증 + AR NPS -2.06 |
| SCIT (풀) | Durham 1999 *NEJM* | 3년 이상 지속 효과, 특이 IgG4↑ |
